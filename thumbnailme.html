Установка  Thumbnail me  в Debian 7

Для  начала устнавливаем необходимые для  работы  программы  пакеты:

apt-get install -y libqt4-core libqt4-gui libqt4-dbus

Установка Thumbnail Me

wget http://sourceforge.net/projects/thumbnailme/files/Linux/Debian/x86/Thumbnailme_2.1_i386.deb

dpkg -i Thumbnailme_2.1_i386.deb

Для 64bit версии OC

wget http://sourceforge.net/projects/thumbnailme/files/Linux/Debian/x64/Thumbnailme_2.1_amd64.deb

dpkg -i Thumbnailme_2.1_amd64.deb
